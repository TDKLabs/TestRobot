
The TestROBOT toolkit contains the necessary tools to allow 
developers and qa folks alike to prototype ideas 
and create tests quickly and efficiently using 
Selenium, SoapUI.

The following tools are included in this kit:

    * Selenium 2.45
    * Python 2.7.2 (RobotFramework)
    * SoapUI (open source edition) 5.2.0, Groovy 2.2.2
